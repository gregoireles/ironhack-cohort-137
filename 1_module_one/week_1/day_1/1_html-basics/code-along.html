HTML basics

HTML is all about structuring your pages.
HTML doesn't care about styling... But browsers prodive some default styling.
HTML is the basic block of your apps, so treat it carefully. You must validate your code !!!
It's simple but not simplistic : hence you can do powerfull things with simple rules !

Consider any HTMl document as a tree.
This document tree has a single root : <html>
The root tag contains 2 children : <head> and <body>
<head> contains all metadata : a mandatory title tag and optional links, scripts, meta, etc.
All these tag are node elements of the document tree.

If it's not a tag, it's a a text node : containing regular text (not hyper-text).
